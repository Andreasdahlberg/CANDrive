# CANDrive
CANDrive is a brushed DC Motor Controller controlled via a [CAN](https://en.wikipedia.org/wiki/CAN_bus) interface.

## Features

* Velocity control
* Torque control
* CAN interface
* Firmware updates over CAN
* Emergency stop 
* 5 A output current
* 12-24 V supply voltage
